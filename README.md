Proyecto ambiental.
Este es un proyecto el cual va a ayudar en la realizacion de proyectos de ciencias naturales, cuenta con sensores de distintos agentes que se encuentran en el aire ensamblados en un prototipo con ESP32.
El backend consiste en una API realizada con FastAPI en Python la cual ayuda a comunicar el prototipo fisico junto a la web en la cual se encontrara el dashboard. Esta API, recibira solicitudes y hara consultas a la base de datos o devolvera distintos objetos dependiendo la consulta realizada a esta misma.
